# Tapias Hookah (React + Vite)

Awwwards-inspired premium landing with:
- Text reveal animations (Framer Motion)
- Magnetic CTA button
- Animated aurora background + subtle smoke canvas
- Mobile-first layout + snap gallery

## Run locally
1) Install Node.js (LTS)
2) In this folder:
```bash
npm install
npm run dev
```
Open the URL shown (usually http://localhost:5173)

## Build for hosting
```bash
npm run build
npm run preview
```

## Customize quickly
- Replace hero/brand images in `public/assets/`
- Update flavors in `src/sections/Flavors.jsx`
- Replace WhatsApp / Maps links in `src/sections/Gallery.jsx` (CTA buttons)
